# 서울 코믹스 계좌 이체 페이지 기획

## 프로젝트 개요

서울 코믹스 행사에서 키링, 사진 등 굿즈를 판매할 때,
구매자가 QR코드를 스캔해 판매자 계좌로 간편하게 이체할 수 있는 정적 HTML 페이지.

GitHub에 공개 저장소로 올리고, GitHub Pages로 배포하여 QR코드 링크로 사용.

---

## 핵심 기능

### 1. 계좌번호 표시 및 복사
- 은행명, 계좌번호, 예금주명 화면에 표시
- "복사" 버튼 클릭 시 계좌번호를 클립보드에 복사
- 복사 성공 시 피드백 메시지 표시 ("복사됨!")

### 2. 은행 앱 바로가기
- 주요 은행 버튼 목록 제공 (카카오페이, 토스, 네이버페이 포함)
- 버튼 클릭 시 해당 은행 앱 딥링크로 실행
- 앱 미설치 시 앱스토어/플레이스토어로 이동 (폴백 처리)

### 3. QR 코드
- 이 페이지 URL을 QR코드로 변환하여 출력용으로 사용
- 구매자가 QR 스캔 → 페이지 접속 → 계좌 확인 및 이체

---

## 기술 스택

| 항목 | 내용 |
|------|------|
| 파일 형태 | 단일 HTML 파일 (`index.html`) |
| 외부 의존성 | 없음 (순수 HTML/CSS/JS) |
| 호스팅 | GitHub Pages |
| 접근 방법 | QR코드 → GitHub Pages URL |

## 제안 사항

### 보안 / 공개 저장소 주의점
- **계좌번호는 HTML에 직접 작성해도 되지만**, 공개 저장소이므로 누구나 볼 수 있음
- 실제 계좌번호 노출이 불편하다면 GitHub Secrets 대신 `.env` 파일을 `.gitignore`에 추가하거나, 계좌번호 일부를 마스킹 처리하는 방법 고려 가능
- 단, 이체 목적상 전체 번호 표시가 필수이므로 공개 노출은 불가피

### UX 개선 제안
1. **금액 입력 칸** — 구매자가 금액을 직접 입력하고, 복사 시 "계좌번호 | 금액: 5,000원" 형식으로 메모 포함 가능
2. **다국어 지원** — 외국인 구매자를 위한 영어/일어 병기 (서울 코믹스 특성상 유용)
3. **상품 목록 표시** — 키링 ○○원, 사진 ○○원 등 가격표를 함께 표시하면 구매자 편의 향상
4. **QR 코드 인쇄용 페이지** — `?print=true` 파라미터로 QR + 계좌정보만 깔끔하게 출력되는 뷰 제공
5. **다크모드** — 행사장 조명 환경에 따라 화면 밝기 조절

### 딥링크 지원 현황
| 앱 | 딥링크 지원 | 비고 |
|----|------------|------|
| 카카오페이 | ✅ | `kakaopay://...` |
| 토스 | ✅ | `supertoss://...` |
| 네이버페이 | ✅ | `naversearchapp://...` |
| 시중 은행 앱 | 일부 ✅ | 은행별로 상이, 앱스토어 폴백 필요 |
